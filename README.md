# 📊 Banking Dashboard Analytics

## 📌 Project Overview
This project is a **Power BI Banking Dashboard** designed to analyze customer financial data and understand **risk analytics in banking**.  
It helps banks decide whether to approve loans based on customer profile and financial behavior.

---

## 🎯 Problem Statement
To develop a basic understanding of **risk analytics in banking** and use customer financial data to minimize the risk of loan default.

---

## 💡 Solution
Using Power BI dashboards, we analyze:

* Customer income & occupation
* Loan distribution
* Deposit trends
* Account balances
* Risk-based decision insights

This helps banks take smarter lending decisions.

---

## 📂 Dataset Information
Dataset includes multiple related tables:

* Banking Relationship
* Client Banking
* Gender
* Investment Advisor
* Period

Columns include:
Client ID, Age, Gender, Income, Occupation, Deposits, Loans, Accounts, Risk Weighting, etc.

---

## 🧹 Data Cleaning & Feature Engineering
* Created **Income Band** column (Low / Mid / High)
* Created **Engagement Days** using DATEDIFF
* Created **Processing Fees** based on Fee Structure
* Created Engagement Timeframe

---

## 📈 KPIs Used
* Total Clients
* Total Loan
* Total Deposit
* Bank Loan
* Business Lending
* Total Fees
* Credit Card Amount
* Saving Account Amount
* Foreign Currency Amount
* Engagement Accounts
